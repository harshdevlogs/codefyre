## ✨ Features

### Interactive "WE BUILD" Box
- **Spotlight / torch effect** — mouse acts as a torch; only what's near the cursor is illuminated
- **"Hover to explore" hint** — frosted blur overlay prompts interaction before the user engages
- **Floating draggable tags** — five stat/achievement tags float with physics inside the box:
  - 🏆 150+ Projects Completed
  - 👥 Highly Experienced Team
  - ⭐ Top-Rated Agency
  - 🌐 Websites at Low Cost
  - 🚀 5+ Years Experience
- **Bounce physics** — tags drift continuously and bounce off the box walls
- **Drag & drop** — tags can be grabbed and repositioned; they bounce off boundaries on drag too
- Touch-friendly: works with finger on mobile/tablet with "Touch to explore" hint

### Navigation
- Fixed navbar with scroll-triggered blur/background effect
- Desktop: full nav links + "Let's Talk" CTA button
- Mobile: hamburger button opens a full-screen overlay menu with large tap targets

### Animations & UX
- Custom cursor with ring follower (desktop only, disabled on touch devices)
- Scroll-reveal animations on all major elements with staggered delays
- Animated hero entrance (staggered fadeUp sequence)
- Animated scroll-line hint in the hero
- Infinite marquee strip
- Ambient glow animation in the About visual box

### Responsive Design
Three fully distinct layout breakpoints:

**Desktop (1025px+)**
- Full two-column layouts, custom cursor, hover-only interactions

**Tablet (641px – 1024px)**
- Tighter nav, services collapse to single column, 2×2 Why Us grid, 2-column portfolio

**Mobile (≤ 640px)**
- Hamburger menu, stacked hero with full-width buttons, reordered About section (text first), single-column layouts throughout, no horizontal scroll
